## Mechanics Notes

### Overview

- Game rules are configuration choices available when starting a new game that can slightly alter some mechanics. Some options will disable Achievements. Game rules can be saved as presets.

### Difficulty

- The difficulty setting will grant either player or AI rulers powerful bonuses. Setting it to Easy or Very Easy disables achievements.

### Gameplay options

- The gameplay options can change the game to be less historical.

### Historical events

- The following game rules determine how certain historical events will play out.

### Conquerors

- Conquerors rules determine how Conquerors affect gameplay and the Conqueror trait. None of them disables achievements.

### Situations

- Situation rules determine whether certain situations will be present in the game or not. None of them disables achievements.

### Extra Administrative Realms

- Extra Administrative Realms rules can make a realm start with Administrative government to better reflect its historical organization. None of them disables achievements.

### Extra Nomadic Region

- Extra Nomadic Region rules can make a region use nomad mechanics. None of them disables achievements.

### Ironman mode

- Ironman mode is a special game rule that disables character switching and manual saving. The game is saved automatically once every 6 months as well as when exiting the game. Prior to version 1.9.X, achievements could only be obtained in Ironman mode. In Ironman mode, the game over screen only has the option to exit to the main menu.
